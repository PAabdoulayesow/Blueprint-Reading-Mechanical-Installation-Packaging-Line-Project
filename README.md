# Blueprint Reading & Mechanical Installation — Packaging Line Project

## Overview
This project is based on an applied automation and mechanical project
completed during my Master's in Automation & Robotics (Université de
Poitiers / SP2MI-CNRS): a full packaging line cell involving mechanical
layout, equipment installation/alignment principles, and GRAFCET-based
control sequencing.

It targets the mechanical/installation side of a **Millwright/Industrial
Mechanic** role: reading blueprints, installing and aligning equipment, and
understanding how mechanical stations interact in a production line —
complementary to, but separate from, the PLC/SCADA automation project and the
maintenance project in this portfolio.

*Note: original academic project files are covered by university NDA
(industry partnership); this repo presents the reformulated, generalized
version of the work and methodology.*

## Scope
- Reading and interpreting mechanical layout drawings/blueprints
- Equipment installation and alignment principles (stationary machinery)
- Foundation and mounting considerations
- Sequential control logic (GRAFCET) governing station-to-station operation
- Hand/power tool and fabrication considerations relevant to installation
